## 🎯 Sobre Mim

Desenvolvedor full stack e empreendedor de tecnologia, fundador da **Kairus** — empresa especializada na criação e desenvolvimento de sistemas personalizados. Estudante de Ciência de Dados e Inteligência Artificial no IBMEC, com foco em soluções reais que combinam produto, dados e IA. Meu stack principal é React, Python/FastAPI e PostgreSQL, com infraestrutura na AWS e integrações como WhatsApp Cloud API, Meta Ads e transcrição por IA.

## 🚀 Projetos em Destaque

### 🏥 [NAPSE](https://napse.app) — SaaS para Clínicas Médicas · *Kairus*
Plataforma vertical para gestão de clínicas e médicos no Brasil, desenvolvida pela Kairus. Integra agenda, prontuário, transcrição por IA (Whisper), WhatsApp Cloud API, Meta Ads e geração de landing pages. Stack: React + FastAPI + PostgreSQL + AWS. Tech Provider verificado pela Meta.

---

### 📊 [Projeto Cloud — Pipeline B3](https://github.com/JoGabTasca/Projeto-Cloud)
Pipeline em nuvem para coleta, transformação e análise de cotações da B3. Arquitetura cloud com foco em dados financeiros em tempo real. Colaboração com equipe IBMEC. ⭐ 1

---

### 🤖 [ML — Machine Learning IBMEC](https://github.com/JoGabTasca/ML)
Repositório com notebooks da disciplina de Machine Learning: preparação de dados, análise exploratória, pipelines de classificação, avaliação de modelos e AutoML.

---

### 💬 [NLP — Processamento de Linguagem Natural](https://github.com/JoGabTasca/NLP)
Notebooks desenvolvidos em Google Colab cobrindo tarefas clássicas de PLN: pré-processamento textual, extração de características, análise de sentimentos e classificação.

---

### 🌆 [Vish — Sistema de Reporte Urbano](https://github.com/JoGabTasca/ProjetoBI-n8n)
Solução para reportar problemas urbanos com interface em React, geolocalização, upload de fotos e automação via n8n. Foco em UX para cidadãos.

---

### ☁️ [BigData Cloud 2025](https://github.com/JoGabTasca/BigDataCloud_2025_2)
